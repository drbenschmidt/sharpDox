Namespace: SharpDox.Sdk.Exporter

Syntax

Beschreibung

Exportplugins müssen dieses Interface implementieren.

Mitglieder

CheckRequirements

Deklariert von: SharpDox.Sdk.Exporter.IExporter

Syntax

public abstract Boolean CheckRequirements()

Beschreibung

Diese Methode wird während des Bauvorgangs aufgerufen, um die Vorraussetzungen des Plugins zu überprüfen.

Rückgabe

True, falls alle Vorraussetzungen erfüllt wurden. Ansonsten false und der Bauvorgang wird gestoppt.

Export

Deklariert von: SharpDox.Sdk.Exporter.IExporter

Syntax

public abstract Void Export(SDRepository repository, String outputPath)

Beschreibung

Diese Methode wird während des Bauvorgangs aufgerufen, um den Exportvorgang anzustoßen.

Parameter

repository

Die eingelesene Lösung.

outputPath

Der ausgabepfad für das Plugin. Alle Dateien, welche durch das Plugin erstellt werden, sollten in diesem Pfad gespeichert werden.
