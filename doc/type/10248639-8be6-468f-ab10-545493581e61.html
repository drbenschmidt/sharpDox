IBuildMessenger

Namespace: SharpDox.Sdk.Build

Syntax

public interface IBuildMessenger

Beschreibung

Dieses Interface bildet alle Events ab, die von der Implementierung des IBuildControllers genutzt werden.

Mitglieder

Name Beschreibung
OnBuildMessage

Wird gefeurt, falls der BuildController eine Nachricht während des Bauvorgangs sendet.
OnBuildProgress

Wird gefeurt, falls der BuildController eine Fortschrittsmeldung für den aktuellen Bauvorgang sendet.
OnBuildStopped

Wird gefeurt, falls die Methode StartBuild des IBuildController gestoppt wurde.
OnParseCompleted

Wird gefeurt, falls die Methode StartParse des IBuildController beendet wurde. Das SDRepository kann null sein, falls das Einlesen nicht erfolgreich war.
OnStepMessage

Wird gefeurt, falls der BuildController eine Nachricht während des aktuellen Vorgangs sendet.
OnStepProgress

Wird gefeurt, falls der BuildController eine Fortschrittsmeldung für den aktuellen Vorgang sendet.
