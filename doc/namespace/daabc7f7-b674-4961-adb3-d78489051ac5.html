SharpDox.Model.CallTree

Assembly: SharpDox.Model

Beschreibung

Die Elemente in diesem Namensraum werden von SDMethod genutzt. Die Eigenschaft Calls enthält alle Aufrufe zu anderen Projekt-Methoden. SDNode ist die Basis aller anderen Typen. Der SDConditionalBlock repräsentiert zum Beispiel ein if Statement, der SDBlock einen Code-Block wie zum Beispiel eine while oder foreach Schleife. SDTargetNode repräsentiert einen Aufruf zu einer anderen Projekt-Methode.

Der Aufrufsbaum enthält nur Aufrufe zu anderen Projekt-Methoden.

Typen

Name Description
SDConditionalBlock

Repräsentiert einen Blockausdruck der eine Bedingung ist.
SDBlock

Repräsentiert einen Blockausdruck der keine Bedingung ist.
SDNode

Basisklasse für alle Elemente im Aufrufsbaums.
SDNodeRole

Alle Rollen die ein Element im Ausdrucksbaum annehmen kann.
SDTargetNode

Repräsentiert einen Methodenaufruf.
