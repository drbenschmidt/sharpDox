SharpDox.Model.Documentation.Token

Assembly: SharpDox.Model

Beschreibung

Der SharpDox.Model.Documentation.Token Namensraum beinhaltet alle Typen die für das Inline-Dokumentations-Modell von sharpDox genutzt werden.

Typen

Name Description
SDCodeToken

Der Code-Token repräsentiert einen Code-Abschnitt einer Inline-Dokumentation.
SDToken

Repräsentiert einen "einfachen" Text in einer Inline-Dokumentation. Außerdem ist es die BAsis-Klasse für SDSeeToken und dem SDCodeToken.
SDTokenList
SDTokenRole

Alle Rollen die ein Element in einer Dokumentation annehmen kann.
SDSeeToken

Repräsentiert eine Referenz innerhalb einer Inline-Dokumentation.
