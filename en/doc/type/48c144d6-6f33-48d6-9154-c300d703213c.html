IBuildMessenger

Namespace: SharpDox.Sdk.Build

Syntax

public interface IBuildMessenger

Summary

This interface exposes all events used by the implementation of the IBuildController.

Members

Name Summary
OnBuildMessage

Gets fired, if the BuildController sends a message during the build process.
OnBuildProgress

Gets fired, if the BuildController sends a progress update for the current build process.
OnBuildStopped

Gets fired, if the StartBuild method of the IBuildController has stopped.
OnParseCompleted

Gets fired, if the StartParse method of the IBuildController has finished. The SDRepository can be null, if the parse wasn't successful.
OnStepMessage

Gets fired, if the BuildController sends a message during the current step of the build process.
OnStepProgress

Gets fired, if the BuildController sends a progress update for the current step of the build process.
