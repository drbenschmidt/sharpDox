IExporter

Namespace: SharpDox.Sdk.Exporter

Syntax

public interface IExporter

Summary

Every plugin which wants to expose export functionality has to implement this interface.

Members

Name Summary
OnStepMessage

The BuildController of sharpDox listens on this event. The Exporter is able to post messages which will displayed in the progressbar.
OnStepProgress

The BuildController of sharpDox listens on this event. The Exporter is able to post a progress which will displayed in the progressbar.
Export

This method gets called by the build process to start the export process.
Author

Name of the author. This will be shown in the exporter selection.
Description
