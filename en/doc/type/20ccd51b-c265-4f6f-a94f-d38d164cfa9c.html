IBuildController

Namespace: SharpDox.Sdk.Build

Syntax

public interface IBuildController

Summary

This interface exposes all needed functionalities to parse and build documentations.

Members

Name Summary
StartBuild

Starts a documentation build for a given c# solution. The method parses the solution and starts all registered exporters.
StartParse

Starts a structure parse for a given c# solution. This method only parses minimal information about namespaces, types and members. Just enough to show a tree of all members as seen in the visibility settings of the sharpDox GUI.
Stop

Stops any current running parse or build process.
