SharpDox.Model.CallTree

Assembly: SharpDox.Model

Description

Types in this namespace are used in the SDMethod type. The property Calls contains a structure of calls to other project methods. SDNode is the base of all other types. The SDConditionalBlock represents a conditional block like a if statement. SDBlock represents blocks like while or foreach loops. SDTargetNode represents the call to a project method.

The calltree only contains calls to project methods.
